# 单调队列（deque）

高效求解「滑动窗口 / 子区间最值」问题的数据结构。单次区间最值查询 $O(1)$。

## deque 常用操作

### 定义

```cpp
template <
	class T,
	class Allocator = std::allocator<T>
> class deque;
```

### 操作

```cpp
reference operator[](size_t pos); // O(1) 大常数

// O(1)
reference front();
reference back(); 

// O(1)
void push_front(const T &value);
void push_back(const T &value);
void pop_front();
void pop_back();
```
